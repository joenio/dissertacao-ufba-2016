138     /*--- Thunk layout                                         ---*/
139     /*------------------------------------------------------------*/
140     
141     /*
142        Z -- value is zero extended to 32 / 64 bit
143        S -- value is sign extended to 32 / 64 bit
144        F -- a binary floating point value
145     
146        +--------------------------------+-----------------------+----------------------+-------------+
147        | op                             |   cc_dep1             |   cc_dep2            |   cc_ndep   |
148        +--------------------------------+-----------------------+----------------------+-------------+
149        | S390_CC_OP_BITWISE             | Z result              |                      |             |
150        | S390_CC_OP_SIGNED_COMPARE      | S 1st operand         | S 2nd operand        |             |
151        | S390_CC_OP_UNSIGNED_COMPARE    | Z 1st operand         | Z 2nd operand        |             |
152        | S390_CC_OP_SIGNED_ADD_32       | S 1st operand         | S 2nd operand        |             |
153        | S390_CC_OP_SIGNED_ADD_64       | S 1st operand         | S 2nd operand        |             |
154        | S390_CC_OP_UNSIGNED_ADD_32     | Z 1st operand         | Z 2nd operand        |             |
155        | S390_CC_OP_UNSIGNED_ADD_64     | Z 1st operand         | Z 2nd operand        |             |
156        | S390_CC_OP_UNSIGNED_ADDC_32    | Z 1st operand         | Z 2nd operand        | Z carry in  |
157        | S390_CC_OP_UNSIGNED_ADDC_64    | Z 1st operand         | Z 2nd operand        | Z carry in  |
158        | S390_CC_OP_SIGNED_SUB_32       | S left operand        | S right operand      |             |
159        | S390_CC_OP_SIGNED_SUB_64       | S left operand        | S right operand      |             |
160        | S390_CC_OP_UNSIGNED_SUB_32     | Z left operand        | Z right operand      |             |
161        | S390_CC_OP_UNSIGNED_SUB_64     | Z left operand        | Z right operand      |             |
162        | S390_CC_OP_UNSIGNED_SUBB_32    | Z left operand        | Z right operand      | Z borrow in |
163        | S390_CC_OP_UNSIGNED_SUBB_64    | Z left operand        | Z right operand      | Z borrow in |
164        | S390_CC_OP_LOAD_AND_TEST       | S loaded value        |                      |             |
165        | S390_CC_OP_LOAD_POSITIVE_32    | S loaded value        |                      |             |
166        | S390_CC_OP_LOAD_POSITIVE_64    | S loaded value        |                      |             |
167        | S390_CC_OP_TEST_AND_SET        | Z tested value        |                      |             |
168        | S390_CC_OP_TEST_UNDER_MASK_8   | Z tested value        | Z mask               |             |
169        | S390_CC_OP_TEST_UNDER_MASK_16  | Z tested value        | Z mask               |             |
170        | S390_CC_OP_SHIFT_LEFT_32       | Z value to be shifted | Z shift amount       |             |
171        | S390_CC_OP_SHIFT_LEFT_64       | Z value to be shifted | Z shift amount       |             |
172        | S390_CC_OP_INSERT_CHAR_MASK_32 | Z result              | Z mask               |             |
173        | S390_CC_OP_BFP_RESULT_32       | F result              |                      |             |
174        | S390_CC_OP_BFP_RESULT_64       | F result              |                      |             |
175        | S390_CC_OP_BFP_RESULT_128      | F result hi 64 bits   | F result low 64 bits |             |
176        | S390_CC_OP_BFP_32_TO_INT_32    | F source              |                      |             |
177        | S390_CC_OP_BFP_64_TO_INT_32    | F source              |                      |             |
178        | S390_CC_OP_BFP_128_TO_INT_32   | F source hi 64 bits   |                      |             |
179        | S390_CC_OP_BFP_32_TO_INT_64    | F source              |                      |             |
180        | S390_CC_OP_BFP_64_TO_INT_64    | F source              |                      |             |
181        | S390_CC_OP_BFP_128_TO_INT_64   | F source hi 64 bits   |                      |             |
182        | S390_CC_OP_BFP_TDC_32          | F value               | Z class              |             |
183        | S390_CC_OP_BFP_TDC_64          | F value               | Z class              |             |
184        | S390_CC_OP_BFP_TDC_128         | F value hi 64 bits    | F value low 64 bits  | Z class     |
185        | S390_CC_OP_SET                 | Z condition code      |                      |             |
186        +--------------------------------+-----------------------+----------------------+-------------+
187     */
188
